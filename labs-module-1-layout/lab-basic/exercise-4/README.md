# Module 1 - Basic Laboratory - Exercise 4

Create a card-type element using CSS Grid.

## Installation to develop

1. Install the Node.js dependencies:
   ```bash
   cd labs-module-1-layout/lab-basic/exercise-4
   npm i --save-dev
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. If you want to validate the SASS code according Stylelint standard only, run:
   ```bash
   npm run lint:scss
   ```
4. End and happy coding!
